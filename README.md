# Sprint_4
# Автотесты для сервиса «Яндекс Самокат»

Проект содержит автоматизированные тесты для проверки функциональности веб-приложения «Яндекс Самокат». Тесты реализованы с использованием паттерна **Page Object Model (POM)**.

## 🛠 Технологии и инструменты


| Технология | Версия | Описание |
| :--- | :--- | :--- |
| **Java** | 11 | Язык программирования |
| **JUnit** | 4.13.2 | Фреймворк для написания тестов |
| **Selenium** | 3.141.59 | Инструмент для автоматизации действий в браузере |
| **WebDriverManager** | 5.6.2 | Библиотека для управления драйверами браузеров |
| **Maven** | 3.9.x (или ваша локальная) | Система сборки проекта |

## 🧪 Что проверяется
1.  **Выпадающий список в разделе «Вопросы о важном»**: проверка соответствия текста ответов выбранным вопросам.
2.  **Заказ самоката**: проверка всего флоу оформления заказа через две точки входа (верхняя и нижняя кнопки «Заказать»).

## 🚀 Запуск тестов
Для запуска всех тестов из терминала используйте команду:
`mvn clean test`

## 📁 Структура проекта
* `src/test/java/pages` — классы Page Object с локаторами и методами взаимодействия.
* `src/test/java/tests` — тестовые классы и базовый класс `BaseTest` для настройки драйвера.
